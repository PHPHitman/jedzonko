## General info
* Application display food offered by restaurants.
* Users can search by clicking buttons with company name or category (eg. drinks), users also can use searching window and type product name.
* If click on product picture it will be automaticly added into personal order. After submit user can delete order or delete single products and save changes, action is possible until status is not changed by admin.
* Admin see all of the orders made by users in current day and can change order status for pending or completed, users see it under order.
* Admin can register new users, manage products and companies.
